# Проект: Web Crawler (C#)

## Технологии
- C#
- .NET 8
- [HtmlAgilityPack](https://www.nuget.org/packages/HtmlAgilityPack)

## Функционалност
- Многостепенен краулинг (по 2 нива по подразбиране)
- Проверка на `robots.txt`
- Съхранение на данни в JSON
- Конзолен интерфейс за преглед на резултатите

## Стъпки за стартиране

1. **Клонирай репозитория:**
```bash
git clone https://github.com/inaD12/Crawler
cd Crawler
```
2. **Инсталирай зависимостта HtmlAgilityPack:**
```bash
dotnet add package HtmlAgilityPack
```
3. **Стартирай приложението:**
```bash
dotnet run
```

4. **Стартирай приложението.**

5. **Прегледай резултатите в конзолата и JSON файловете в папка data/.**
